# 周报管理

每周交付物按周归档，每周包含三类文件：

## 目录结构

```
第N周/
├── 周计划/      # 小组周计划（每周1份，组长汇总）
├── 周报/        # 个人周报（每人1份）
└── 会议纪要/    # 小组会议记录（每周1-2份）
```

## 文件命名规范

| 类型 | 命名格式 | 示例 |
|------|----------|------|
| 周计划 | `第N周周计划.doc` | `第1周周计划.doc` |
| 个人周报 | `第N周周报_姓名.doc` | `第2周周报_张三.doc` |
| 会议纪要 | `第N周小组会议记录M.doc` | `第1周小组会议记录1.doc` |

## 模板位置

所有模板在 `docs/模板/` 目录下：
- `周计划模板 .doc`
- `夏季小学期周报模板第1-4周_姓名.doc`
- `小组会议记录模板.doc`
