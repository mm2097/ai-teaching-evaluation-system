## 一、总体判断

- **真正扎实的部分**：认证登录、题库 CRUD、AI 出题主流程（SSE 流式）、AI 判题、Agent 工具调用、算法服务（线性回归 / 预警规则 W1/W2/W3/W5 / 评价加权 / TF-IDF 去重）——这些是真实后端 + 真实 LLM 调用。
