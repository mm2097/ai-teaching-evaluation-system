## 0. 优先级与工程量约定

### 优先级
- **P0**：算法已就绪（边际成本低）或核心差异化（不做系统失色）—— 优先冲刺
- **P1**：核心能力基石 / 闭环器（不做其他能力塌方）
- **P2**：体验优化 / 增强项（可延后）

### 工程量
- **S（小）**：算法/接口已就绪，只欠前端聚合或少量封装，1–2 天
- **M（中）**：需新接口或新算法封装，3–5 天
- **L（大）**：全新能力（无现成算法），需设计+实现+调优，1 周以上
